# Maler Philipp Bühler GmbH - Website Redesign

## Style DNA

### Mood (3 Adjectives)
- **Präzise** - Clean lines, deliberate spacing, exact craftsmanship
- **Ruhig** - Calm color palette, generous whitespace, unhurried rhythm  
- **Hochwertig** - Premium materials, refined typography, tactile details

### Typography Pairing
- **Headings**: Georgia/Cambria (Editorial serif) - warm, trustworthy, craft-oriented
- **Body**: Inter (Clean grotesk) - modern, highly legible, Swiss precision

### Layout Rhythm
- Big typographic hero with strong hierarchy
- Alternating image/text sections with generous whitespace
- Full-bleed moments balanced with contained content
- Consistent 80-120px section spacing

### Craft Cues
- Subtle paper texture overlay (opacity 0.03)
- Thin brush stroke SVG dividers between sections
- Warm terracotta/sienna brand color (earthy painter tones)
- Corner-frame motif for image containers
- Quality stamp micro-labels: "20+ Jahre Erfahrung", "Ur-Rooter"

### Signature Element
- Thin "brush stroke" SVG line between sections
- Subtle parallax on editorial images
- Material swipe gallery with texture reveal on hover

### Color System
```
--paper: Warm off-white #FAF8F5
--ink: Deep warm black #231F1B  
--brand: Terracotta/Sienna #A67C5B
--brand-light: Light terracotta tint #F5EDE6
--brand-dark: Darker terracotta #8B6346
--muted: Muted warm gray #7A726A
--line: Subtle border #D9D4CE
```

### Interaction Design
- Section reveals on scroll (Framer Motion)
- Text stagger animations
- Image parallax on scroll
- Hover micro-interactions on cards
- Material swipe gallery signature interaction
